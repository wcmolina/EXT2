# EXT2
Java implementation of a simplified EXT2 file system structure in a 256 MB binary file. This was a project for Operating Systems II.

## Implemented commands
* ls
* ls -l
* mkdir [dirname]
* cat [filename]
* cat > [filename]
* cat >> [filename]
* rm [filename]
* rmdir [dirname]
* cd [path]

## Incomplete commands
* ln [source] [dest]
* ln -s [source] [dest]
